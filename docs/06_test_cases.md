# Задание 4.1: Тест-кейсы

## Промпт для ИИ

```text
Ты — Senior QA Engineer. На основе реализованных фич (Просмотр слотов, Бронирование, Отмена брони) и требований из файла 01_analytics_requirements.md, сгенерируй 7 подробных тест-кейсов.

Оформи их в виде Markdown-таблицы со столбцами:
| ID | Название | Предусловия | Шаги | Ожидаемый результат |

Учти граничные случаи: попытка записать 9-го новичка, отмена за 1 час до старта, попытка записаться на отменённый центром слот, фильтрация дат с "С" больше "По".

```

|  ID  | Название | Предусловия | Шаги | Ожидаемый результат |
|------|----------|-------------|------|--------------|
|TC-01|Просмотр слотов по умолчанию|Пользователь открыл приложение|1. Открыть раздел «Расписание»|Отображаются слоты на 7 дней вперёд (FR-01.1). Каждая карточка содержит: время старта, длительность, тип трассы, маршала, свободные места, статус экипировки, цену (FR-01.2).|
|TC-02|Фильтрация слотов по датам|Пользователь в разделе «Расписание»| 1. Выбрать дату «С» = завтра; 2. Выбрать дату «По» = через 3 дня; 3. Нажать «Применить»|Отображаются только слоты в выбранном диапазоне дат (FR-01.3)|
|TC-03|Бронирование с прокатом экипировки|Пользователь выбрал слот со статусом «available»|1. Нажать «Записаться»; 2. Выбрать «Прокат» экипировки; 3. Выбрать уровень «Новичок»; 4. Нажать «Подтвердить»| Открывается модалка. Цена динамически увеличивается на стоимость проката (FR-04.1). После подтверждения — экран «Запись подтверждена» с деталями и toast-уведомление «📩 Запись подтверждена» (FR-02.1, FR-02.2).
|TC-04|Ошибка «Лимит новичков превышен»|Пользователь выбрал слот, бэкенд вернул ошибку NOVICE_LIMIT_EXCEEDED|1. Нажать «Записаться»; 2. Выбрать уровень «Новичок»; 3. Нажать «Подтвердить»|Появляется сообщение «На этот заезд уже записалось слишком много новичков. Выберите другой слот или укажите уровень «Опытный»» (FR-02.3, NFT-02).
|TC-05|Отмена брони за 3 часа до старта|У пользователя есть активная бронь, до старта которой 3 часа|1. Перейти в «Мои заезды»; 2. Найти активную бронь; 3. Нажать «Отменить»; 4. Подтвердить в диалоге|Бронь получает статус cancelled_by_client, отображается с серой плашкой «Отменено вами». Появляется сообщение «✓ Бронирование успешно отменено» (FR-03.1, UC-02).
|TC-06|Попытка отмены брони за 1 час до старта|У пользователя есть активная бронь, до старта которой 1 час|1. Перейти в «Мои заезды»; 2. Найти активную бронь|Кнопка «Отменить» заблокирована с подписью «Нельзя отменить». Под ней подсказка «Отмена возможна не позднее чем за 2 часа до старта» (FR-03.2).
|TC-07|Попытка записаться на отменённый центром слот|В расписании есть слот со статусом cancelled_by_center|1. Открыть «Расписание»; 2. Найти отменённый слот|Слот отображается с красной плашкой «Отменён центром: [причина]» (например, «Сильный дождь»). Кнопка «Записаться» заблокирована с текстом «Отменён» (FR-03.3, US-07).
